# Affan Shaikh - Portfolio Website

A personal portfolio website built with Node.js, Express, and EJS for Assignment 2.

## Live Site

Deployed on Render: [(https://affan-portfolio-swwp.onrender.com)]

## Overview

This site presents information about my background, interests, and developing skills
in networking and cybersecurity.

## Features

- Multi-page layout (Home, About, Projects, Contact)
- Shared navigation bar and footer using EJS partials
- Responsive design using Bootstrap
- Contact page with email, GitHub, and LinkedIn links

## Technologies Used

- Node.js
- Express.js
- EJS (Embedded JavaScript Templates)
- Bootstrap (via CDN)
- Font Awesome (via CDN)
- Custom CSS
